📱 使用说明

  1. 自动主题检测: 页面会自动检测你的系统主题设置
  2. 实时切换: 改变系统主题设置时，页面会自动跟随变化
  3. React应用: 点击下方按钮访问主React应用查看主题效果
🌟 访问 React 应用 (http://localhost:3000)

💡 主题切换测试方法

macOS: 系统偏好设置 → 通用 → 外观 → 选择"浅色"或"深色"
Windows: 设置 → 个性化 → 颜色 → 选择应用模式
浏览器: 开发者工具 → 更多工具 → 渲染 → 模拟CSS媒体功能
